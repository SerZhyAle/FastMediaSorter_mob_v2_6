1. **Tap on the folder** you just added (or long-press)
2. You'll see **all your media files** as thumbnails

## Step 4: Sort Files Quickly (2 minutes)

**Setup Quick Sort folders:**

1. Go to **Settings** â†’ **Quick Sort** tab
2. Tap **"Add to Quick Sort"**
3. Select folders where you want to sort files (e.g., "Best Photos", "Vacation")
4. **Each folder gets a number** (0-9) and color

**Now you can sort files instantly:**

1. While viewing a photo, **tap bottom-left corner** (COPY zone)
2. **OR tap the "1" button** on the command panel
3. File is copied to your Quick Sort folder #1!
